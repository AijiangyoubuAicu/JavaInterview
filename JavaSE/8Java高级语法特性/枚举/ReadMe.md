# 枚举

## 概念

枚举Enumeration 

- 最简单的枚举：是一系列具有名称的**常量**
- 在JAVA中，枚举定义了一种**类类型**。通过将枚举定义为类，可具有构造函数、方法或实例变量

创建枚举需要关键字`enum`

定义一个最简单的枚举

```java
/**
 * 一个水果的枚举，列出了四种水果
 */
public enum Fruit {

    /**
     * 包含着"苹果"、"西瓜"、"梨"、"桃子"四个常量
     */
    APPLE, WATERMELON, PEAR, PEACH
}
```

"苹果"、"西瓜"、"梨"、"桃子"等这些**枚举元素**都被称为**枚举常量**；这些常量被称为**自类型化的
self-typed**，**自**是指封装常量的枚举，每个枚举常量被隐式声明为 Fruit
的**公有的**、**静态的**、**final的**成员
